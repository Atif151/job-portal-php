\# Job Portal System



A PHP-based job portal application.



\## Setup Instructions



1\. Clone this repository

2\. Copy `database.example.php` to `database.php`

3\. Update `database.php` with your MySQL credentials

4\. Create database named `find\_jobs` in phpMyAdmin

5\. Run on your localhost



\## Requirements

\- PHP 7.4+

\- MySQL

\- XAMPP/WAMP

"# job-portal-php"
